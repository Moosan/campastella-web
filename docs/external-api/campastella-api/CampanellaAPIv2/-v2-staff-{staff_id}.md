# /v2/staff/{staff_id}

## GET

指定した ID のスタッフを取得する

| パラメータ名 | データ型 | 必須かどうか | 備考 |
| ------------ | -------- | ------------ | ---- |
|              |          |              |      |

## レスポンス

- 200
  - 取得成功
- 401
  - 認証トークンが不正
- 403
  - 権限がない
    - 権限レベル 0 以上が必要
- 404
  - 対象のスタッフが存在しない

### 応答フォーマット

```
{
  "code": 200,
  "response": {
    "staff": {
      "id": 1,
      "discord_id": 123456789012345,
      "discord_name": "",
      "vrc_display_names": [],
      "activated": "yes",
      "updated": "2024-03-20 19:18:52",
      "created": "2024-03-20 19:18:52"
    }
  }
}
```

## PUT

指定した ID のスタッフを更新する

| パラメータ名 | データ型 | 必須かどうか | 備考                          |
| ------------ | -------- | ------------ | ----------------------------- |
| discord_id   | int      | no           | Discord の ID(18 桁の数字)    |
| discord_name | string   | no           | Discord の表示名              |
| activated    | bool     | no           | 有効かどうか(省略時は true)   |
| role         | string   | no           | normal か customer のどちらか |

## レスポンス

- 200
  - 更新成功
- 400
  - リクエスト形式が不正(型不一致など)
- 401
  - 認証トークンが不正
- 403
  - 権限がない
    - 権限レベル 10 以上が必要
- 404
  - 対象のスタッフが存在しない
- 409
  - データ衝突エラー
    - 同一の discordId が既に登録されている

### 応答フォーマット

```
{"code":200,"response":{}}
```
