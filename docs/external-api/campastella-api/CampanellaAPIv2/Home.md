# CampanellaAPI へようこそ！

ここでは API の仕様を記載します。
内容は不定期に更新されます。

## サポート状況

現在は[[/v2]]系をサポートしています。[[/v1]]は削除されました。

### v1 からの変更点

- エンドポイントにおいて、Authorization ヘッダによる Bearer 認証が必須になりました
- より正確なバリデーション機能が追加されました
- エンドポイントのパスが見直され、REST API としてより正確になりました
- 応答フォーマットが見直され、外部キーとして紐づくデータ自体も返却されるようになり、複数のエンドポイントを叩く必要がなくなりました
