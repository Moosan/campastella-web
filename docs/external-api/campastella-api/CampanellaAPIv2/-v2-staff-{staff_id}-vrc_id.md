# /v2/staff/{staff_id}/vrc_id

## GET

指定したスタッフ ID に紐づく VRCID(displayname)を取得する

| パラメータ名 | データ型 | 必須かどうか | 備考 |
| ------------ | -------- | ------------ | ---- |

## レスポンス

- 200
  - 取得成功
- 401
  - 認証トークンが不正
- 403
  - 権限がない
    - 権限レベル 0 以上が必要
- 404
  - 対象のスタッフが存在しない

### 応答フォーマット

```
{
  "code": 200,
  "response": {
    "vrc_display_names": ["aaa", "bbb"]
  }
}
```

## POST

指定したスタッフ ID に VRCID を紐付ける

| パラメータ名 | データ型 | 必須かどうか | 備考  |
| ------------ | -------- | ------------ | ----- |
| display_name | string   | yes          | VRCID |

## レスポンス

- 201
  - 作成成功
- 400
  - 必須パラメータが存在しない / リクエスト形式が不正(型不一致など)
- 401
  - 認証トークンが不正
- 403
  - 権限がない
    - 権限レベル 10 以上が必要
- 409
  - データ衝突エラー
    - リクエストされた VRCID が既に何らかのスタッフに紐付いている

### 応答フォーマット

```
{"code":201,"response":{}}
```

## DELETE

指定したスタッフ ID の VRCID の紐付けを解除する (この操作はキャンセルできません)

| パラメータ名 | データ型 | 必須かどうか | 備考  |
| ------------ | -------- | ------------ | ----- |
| display_name | string   | yes          | VRCID |

## レスポンス

- 200
  - 削除成功
- 401
  - 認証トークンが不正
- 403
  - 権限がない
    - 権限レベル 10 以上が必要
- 404
  - 対象の VRCID が存在しない

### 応答フォーマット

```
{"code":200,"response":{}}
```
