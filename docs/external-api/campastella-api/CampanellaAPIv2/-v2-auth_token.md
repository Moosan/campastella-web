# /v2/auth_token

## POST

新規認証トークン発行

| パラメータ名     | データ型 | 必須かどうか | 備考       |
| ---------------- | -------- | ------------ | ---------- |
| permission_level | int      | yes          | 権限レベル |

## レスポンス

- 201
  - 作成成功
- 400
  - 必須パラメータが存在しない / リクエスト形式が不正(型不一致など)
- 401
  - 認証トークンが不正
- 403
  - 権限がない
    - 権限レベル 255 以上が必要

### 応答フォーマット

新規生成されたトークンが返却されます。表示されるのは 1 度きりです。

```
{"code":201,"response":{"generated_token":"<生成されたトークン>"}}
```

## DELETE

トークン失効

| パラメータ名 | データ型 | 必須かどうか | 備考         |
| ------------ | -------- | ------------ | ------------ |
| token        | string   | yes          | 認証トークン |

## レスポンス

- 200
  - 削除成功
- 400
  - 必須パラメータが存在しない / リクエスト形式が不正(型不一致など)
- 401
  - 認証トークンが不正
- 403
  - 権限がない
    - 権限レベル 255 以上が必要

### 応答フォーマット

```
{"code":200,"response":{}}
```
