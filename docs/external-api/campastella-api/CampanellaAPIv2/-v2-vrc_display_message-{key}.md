# /v2/vrc_display_message/{key}

## GET

指定した Key の Content を取得する

| パラメータ名 | データ型 | 必須かどうか | 備考 |
| ------------ | -------- | ------------ | ---- |

## レスポンス

- 200
  - 取得成功
- 401
  - 認証トークンが不正
- 403
  - 権限がない
    - 権限レベル 0 以上が必要
- 404
  - 対象の Key が存在しない

### 応答フォーマット

```
{
  "code": 200,
  "response": {
    "vrc_display_message": {
      "test": "hoge"
    }
  }
}
```

## PUT

Key を追加・更新する

| パラメータ名 | データ型 | 必須かどうか | 備考               |
| ------------ | -------- | ------------ | ------------------ |
| key          | string   | yes          | 識別キー           |
| content      | string   | yes          | Key に紐づく文字列 |

## レスポンス

- 200
  - 更新成功
- 201
  - 作成成功 (Key が存在しない場合)
- 400
  - リクエスト形式が不正(型不一致など)
- 401
  - 認証トークンが不正
- 403
  - 権限がない
    - 権限レベル 10 以上が必要

### 応答フォーマット

```
{"code":200,"response":{}}
```

- 401

- 403
  - 権限がない
    - 権限レベル 10 以上が必要

### 応答フォーマット
